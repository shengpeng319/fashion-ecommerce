# Design Specification

> 风格关键词：**时尚电商、极简产品展示、质感留白、移动优先**
> 参考气质：Magazzino Massivo / 日系精选电商 / 小众设计师品牌官网
> 设计语言：干净白底 + 大图产品 + 微妙层次 + 购物引导

---

## 1. 色彩系统

### 主色板（Light Theme — 默认）

```
背景层级:
  --bg-primary:    #FFFFFF        页面主背景
  --bg-secondary:  #F5F5F5        次级背景（分类导航、区块分隔）
```

## 2. 排版系统

### 字体

```css
/* 中文优先 */
--font-cn: 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif;

/* 西文 — 系统无衬线 */
--font-sans: -apple-system, BlinkMacSystemFont, 'SF Pro Text', 'Helvetica Neue', sans-serif;

/* 品牌衬线（仅用于品牌Logo / 大标题装饰） */
--font-display: 'Playfair Display', 'Cormorant', 'Georgia', serif;

/* 等宽（价格、编号） */
--font-mono: 'SF Mono', 'Menlo', monospace;
```

### 字号比例

| Token        | Size    | Weight | Line-height | Letter-spacing | 用途                |
|-------------|---------|--------|-------------|----------------|---------------------|
| `text-hero` | 28–32px | 700    | 1.2         | 0              | 首屏活动标题         |
| `text-h1`   | 22px    | 600    | 1.3         | 0              | 页面主标题           |
| `text-h2`   | 18px    | 600    | 1.35        | 0              | 区块标题、分类名      |
| `text-h3`   | 16px    | 500    | 1.4         | 0              | 卡片标题、商品名      |
| `text-body` | 14px    | 400    | 1.6         | 0              | 正文段落、商品描述     |
| `text-sm`   | 12px    | 400    | 1.5         | 0.02em         | 辅助文字、标签        |
| `text-xs`   | 10px    | 400    | 1.4         | 0.02em         | 最小文字（已售、标签） |

### 价格排版

```
价格是电商页面的核心元素，需要特殊处理：
- 现价: 18–22px, 600, var(--accent) 红色
- 原价: 12px, 400, var(--text-quaternary) #999, text-decoration line-through
- 折扣标签: 10px, 500, 白字红底, padding 2px 6px, border-radius 2px
```

### 排版原则

- **标题简洁**：最多两行，不堆砌文字
- **商品名一行截断**：text-overflow ellipsis，不换行
- **价格永远醒目**：红色现价 + 灰色删除线原价
- **中英文之间加空格**：提升可读性

## 3. 间距系统

基于 **4px 网格**（移动端紧凑）：

```
--space-0:   0
--space-1:   4px      图标与文字间距
--space-2:   8px      紧凑元素间距、网格gap
--space-3:   12px     小间距
--space-4:   16px     默认间距、页面左右padding
--space-5:   20px     中等间距
--space-6:   24px     区块内间距
--space-8:   32px     区块标题上方间距
--space-10:  40px     大区块分隔
--space-12:  48px     页面级间距
--space-16:  64px     首屏留白
```

### 间距原则

- **页面两侧 padding 16px**（移动端），内容不贴边
- **商品网格 gap 8px**，紧凑但不挤
- **区块间距 32–48px**，移动端不需要过大留白
- **卡片内间距 12–16px**

## 4. 布局规范

### 容器

```
--container-mobile:  100vw       手机端全宽
--container-tablet:  768px       平板
--container-desktop: 1200px      桌面（电商不需要过宽）
```

### 核心布局模式

| 模式 | 描述 | 适用 |
|------|------|------|
| **全屏轮播Banner** | 100vw × ~45vh，商品主题图+文字叠层 | 首页Hero、活动页 |
| **左右分栏** | 左侧分类导航(25%) + 右侧商品流(75%) | 分类页 |
| **双列网格** | 2 × 1fr + gap 8px | 商品列表（移动端） |
| **三列网格** | 3 × 1fr + gap 8px | 商品列表（桌面端） |
| **瀑布流** | column-count 2, gap 8px | 穿搭灵感、精选集 |
| **单列信息流** | 全宽纵向排列 | 商品详情、内容页 |
| **四宫格** | 2×2 grid, gap 4px | 模特多角度展示 |

## 5. 图片规范（核心视觉策略）

### 商品图原则

```
1. 图片质量: 高清实拍，自然光线，不过度滤镜
2. 图片类型:
   - 模特图: 展示穿着效果，场景化（室内/街头）
   - 平铺图: 展示版型和材质细节
   - 细节图: 面料纹理、五金配件、缝线
3. 比例:
   - 商品卡片图: 1:1 正方形（统一整齐）
   - Banner图: 16:9 或 3:4
   - 模特展示: 3:4 竖版
4. 处理:
   - 圆角: 4px（小图）/ 8px（大图）
   - 不加边框
   - 不加阴影（保持干净）
   - 懒加载 + blur占位
```

### 模特展示区（商品详情页）

```
四宫格布局:
  grid-template: 1fr 1fr / 1fr 1fr
  gap: 4px
  展示: 正面 / 侧面 / 细节 / 动作
  每格 3:4 竖版比例
  整体宽度占页面 55-60%（左侧）
```

## 6. 组件规范

### 通用

```
圆角:
  --radius-xs:     2px       折扣标签
  --radius-sm:     4px       商品卡片、搜索框
  --radius-md:     8px       按钮、大卡片
  --radius-lg:     12px      弹窗、底部Sheet
  --radius-full:   9999px    胶囊按钮、头像

过渡:
  --transition-fast:   150ms ease    微交互
  --transition-base:   250ms ease    常规过渡
  --transition-slow:   400ms ease    页面切换
```

### 底部导航栏（移动端核心）

```
结构:     固定底部，4个Tab
高度:     50px + safe-area-inset-bottom
背景:     #FFFFFF + --shadow-nav
Tab项:    图标(24px) + 文字(10px) + 间距4px
图标颜色: 未选中 #999999 / 选中 #1A1A1A
选中态:   图标填色 + 文字加粗500
Tab:      首页 / 分类 / 购物车 / 我的
购物车:   右上角红色Badge显示数量
```

### 搜索栏

```
高度:     36px
背景:     var(--bg-tertiary) #EFEFEF
圆角:     18px（胶囊形）
左侧:     搜索图标(16px, #999) + 间距8px
占位文字: 14px, #999
右侧:     可选筛选按钮
内间距:   8px 16px
```

### Banner轮播

```
宽度:     100%（减去两侧padding 16px）
高度:     ~45vh 或 固定 180px
圆角:     8px
指示器:   底部居中，小圆点，当前项实色
文字叠层: 底部渐变遮罩
  background: linear-gradient(to top, rgba(0,0,0,0.5) 0%, transparent 60%)
标题:     白色, 18px, 600
副标题:   白色80%, 12px, 400
```

### 商品卡片

```
结构:
  上半: 商品图（1:1正方形, border-radius 4px）
  下半: padding 8px–12px
    - 商品标题: 14px, 500, #1A1A1A, 一行截断
    - 描述/标签: 12px, 400, #999（可选）
    - 价格行: 现价(红色18px 600) + 原价(灰色12px line-through)
    - 促销标签: 红底白字, 10px, padding 2px 4px

悬停（桌面端）:
  图片 scale(1.02) + overflow hidden
  阴影加深
  200ms ease

选中/加购:
  右下角 "+" 图标按钮
  已加购显示数量Badge
```

### 促销横条

```
背景:     var(--accent) #C8102E
文字:     #FFFFFF
高度:     auto, padding 10px 16px
内容:     "购买全套商品" + 已售数量 + 价格 + "一键加购>"
价格:     20px, 700
按钮文字: 14px, 500
圆角:     4px（整体作为CTA区域）
```

### 顶部导航栏

```
高度:     44px（不含状态栏）
背景:     #FFFFFF
左侧:     返回箭头(24px, #1A1A1A)
中间:     页面标题(16px, 600, #1A1A1A, 居中)
右侧:     更多(...) + 搜索(🔍), 间距12px
底部:     1px solid #F0F0F0 分隔线
```

## 7. 动效规范

### 原则：**流畅、自然、不打扰购物流程**

```
页面切换:
  - 进入: 从右滑入 300ms
  - 返回: 向右滑出 250ms
  - 缓动: ease-out

Banner轮播:
  - 自动播放间隔: 4秒
  - 切换: 水平滑入 400ms ease

商品图切换:
  - 缩略图点击: 淡入 200ms
  - 左右滑动: 跟手

加入购物车:
  - 按钮缩放: scale(0.95) 100ms → scale(1) 100ms
  - Badge弹跳: scale(0) → scale(1.2) → scale(1) 300ms

下拉刷新:
  - 原生弹性滚动
  - 刷新指示器旋转

商品列表加载:
  - 骨架屏（灰色块闪动）
  - 渐显 300ms
```

## 9. 响应式断点

```
移动端优先（电商主要流量来自手机）:
  xs:  0        小手机
  sm:  375px    标准手机（iPhone）
  md:  768px    平板（切换为多列、侧边导航）
  lg:  1024px   平板横屏 / 小桌面
  xl:  1280px   桌面

策略:
  - xs-sm: 单列/双列，底部Tab导航
  - md:    双列/三列，左侧分类导航出现
  - lg+:   三列/四列网格，顶部水平导航替代底部Tab
```

## 11. 禁止事项

```
❌ 不用花哨动效（电商核心是效率，不是炫技）
❌ 不用暗色主题做默认（白底是电商标准，用户信任感更高）
❌ 不用过粗边框（用微妙阴影和间距分层）
❌ 不用大段文字（商品描述精简到2-3行）
❌ 不用 emoji 在标题和按钮中
❌ 不用占位图（用真实商品图）
❌ 不用过大的圆角（电商卡片保持方正，4-8px足够）
❌ 不用纯黑 #000 文字（用 #1A1A1A 更柔和）
❌ 不用超过3种强调色（红色足矣）
❌ 不隐藏价格信息（价格永远可见醒目）
❌ 不用无限滚动替代分页时不加骨架屏
❌ 不遮挡商品图（文字叠层只用在Banner，不在商品卡片上）
❌ 不使用平台不兼容的 CSS（确保 H5 和小程序双端一致）
❌ 不使用 Next.js / React 生态组件（项目使用 uni-app + Vue 3）
```

## 12. 高级感 Checklist

每次交付前检查：

```
□ 商品图是否清晰、统一比例、无水印
□ 价格是否红色醒目 + 原价删除线
□ 商品卡片间距是否均匀（8px gap）
□ 搜索栏是否显眼且易用
□ 购物车Badge是否正确显示
```
