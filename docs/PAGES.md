# 页面清单

## Tab 页面 (底部导航)

### 1. 首页 (pages/index/index)
- 搜索栏
- 轮播图
- 分类图标（8个主分类）
- 热销推荐商品
- 活动专区

### 2. 分类 (pages/category/index)
- 左侧分类导航
- 右侧商品网格
- 二级分类支持

### 3. 购物车 (pages/cart/index)
- 商品列表（图片、名称、规格、价格、数量）
- 全选/单选
- 数量修改
- 删除商品
- 结算底部栏

### 4. 个人中心 (pages/user/index)
- 用户信息/头像
- 会员卡片
- 订单入口（待付款/待发货/待收货/已完成）
- 功能菜单（收货地址/会员中心/批发商后台）

## 子包页面

### 订单模块 (subPackage: pages/order)
- **create** - 提交订单（地址选择/商品确认/支付方式/提交）
- **list** - 订单列表（全部/待付款/待发货/待收货/已完成）
- **detail** - 订单详情（状态/物流/商品/金额/操作）

### 会员模块 (subPackage: pages/member)
- **benefits** - 会员权益（等级/折扣/积分规则）

### 收货地址 (subPackage: pages/address)
- **list** - 地址列表
- **edit** - 新增/编辑地址

### 批发商模块 (subPackage: pages/wholesale)
- **index** - 批发首页（商品列表/额度显示）
- **order** - 批量下单（数量选择/备注/提交）
- **orders** - 批发订单（待审核/已通过/已拒绝）

## 功能页面

### 商品详情 (pages/goods/detail)
- 图片轮播
- 价格信息
- 规格选择
- 加入购物车/立即购买
- 商品详情（富文本）
- 推荐商品

### 商品搜索 (pages/search/index)
- 搜索历史
- 热门搜索
- 搜索结果
- 筛选/排序

### 支付结果 (pages/pay/result)
- 支付成功/失败
- 订单号显示
- 返回按钮
